## UQ64x64

Implements fixed point numbers in Move language using the Q number format.

Similar one used in [Uniswap](https://github.com/Uniswap/v2-core/blob/master/contracts/libraries/UQ112x112.sol) v2 core.

## Compile

    aptos move compile
